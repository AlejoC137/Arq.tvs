================================================================================
                    03_ANTEPROYECTO - DESARROLLO DETALLADO DEL DISEÑO
================================================================================

PROPÓSITO:
Esta carpeta contiene el desarrollo detallado del diseño arquitectónico,
incluyendo esquemas técnicos, especificaciones preliminares y documentación
más precisa para validación final antes del proyecto ejecutivo.

================================================================================
                                   SUBCARPETAS
================================================================================

03_01_DOCUMENTOS/
FUNCIÓN: Memoria descriptiva del anteproyecto
CONTENIDO REQUERIDO:
• Anteproyecto_Memoria.pdf - Documento técnico completo
  └─ Incluir: descripción arquitectónica, criterios de diseño, 
      justificación técnica, cumplimiento normativo

03_02_PLANOS/
FUNCIÓN: Planos técnicos del anteproyecto organizados por especialidad

├─ 03_02_01_Arquitectonicos/
   • Anteproyecto_Plantas_Cortes.dwg - Plantas, cortes y fachadas
     └─ Incluir: dimensiones precisas, niveles, materiales indicativos

├─ 03_02_02_Esquemas_tecnicos/
   • Esquema_Electrico.dwg - Distribución eléctrica básica
   • Esquema_Interiorimos.dwg - Propuesta de interiorismo
   • Esquema_Hidrosanitario.dwg - Distribución de instalaciones
   • Esquema_HVAC.dwg - Sistema de climatización
     └─ Todos en fase esquemática, no definitivos

└─ 03_02_03_Esquemas_Paisajismo/
   • Planta_Paisajismo.dwg - Diseño de jardines y exteriores

03_03_IMAGENES/
FUNCIÓN: Imágenes de presentación del anteproyecto
CONTENIDO REQUERIDO:
• Anteproyecto_Renders.png - Renders fotorrealistas del proyecto
  └─ Múltiples vistas: exterior, interior, perspectivas aéreas

03_04_MODELOS_3D/
FUNCIÓN: Modelo tridimensional detallado
CONTENIDO REQUERIDO:
• Anteproyecto_Modelo.skp - Modelo 3D completo y detallado
  └─ Incluir: texturas, iluminación, contexto, mobiliario básico

03_05_ESPECIFICACIONES/
FUNCIÓN: Especificaciones técnicas preliminares
CONTENIDO REQUERIDO:
• Especificaciones_Anteproyecto.xlsx - Listado de materiales y sistemas
  └─ Incluir: acabados, sistemas constructivos, equipos principales

================================================================================
                                  INSTRUCCIONES
================================================================================

PROCESO DE DESARROLLO:
1. Refinar la idea básica aprobada
2. Desarrollar plantas arquitectónicas detalladas
3. Crear cortes y fachadas con proporciones definitivas
4. Esquematizar sistemas técnicos (eléctrico, sanitario, HVAC)
5. Definir criterios de interiorismo y paisajismo
6. Generar renders de calidad de presentación
7. Elaborar memoria descriptiva completa
8. Preparar especificaciones técnicas preliminares

COORDINACIÓN INTERDISCIPLINARIA:
- Arquitectura como base para todas las especialidades
- Validación de esquemas técnicos con especialistas
- Integración de sistemas en el modelo 3D
- Verificación de viabilidad constructiva

CRITERIOS DE CALIDAD:
- Planos a escala y dimensionados correctamente
- Renders que representen fielmente el diseño
- Memoria técnica completa y coherente
- Especificaciones alineadas con el presupuesto

================================================================================
ENTREGABLES PARA APROBACIÓN:
✓ Set completo de planos arquitectónicos
✓ Esquemas de todas las instalaciones técnicas
✓ Renders de calidad de presentación
✓ Memoria descriptiva aprobada
✓ Especificaciones técnicas validadas
✓ Modelo 3D coordinado con todos los sistemas
================================================================================
