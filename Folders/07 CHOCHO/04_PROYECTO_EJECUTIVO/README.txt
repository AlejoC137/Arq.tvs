================================================================================
                                  INSTRUCCIONES
================================================================================

DESARROLLO DEL PROYECTO EJECUTIVO:
1. Revisar y aprobar anteproyecto completamente
2. Coordinar con todas las especialidades técnicas
3. Resolver interferencias entre sistemas
4. Definir especificaciones técnicas finales
5. Crear detalles constructivos críticos
6. Preparar documentación para licitación/construcción
7. Obtener aprobaciones oficiales necesarias

COORDINACIÓN BIM/3D:
- Modelo coordinado con todas las especialidades
- Clash detection ejecutado y resuelto
- Extracción automática de planos desde modelo 3D
- Cuantificación de materiales desde modelo

CONTROL DE CALIDAD:
- Revisión técnica por especialistas senior
- Verificación de cumplimiento normativo
- Validación de constructabilidad
- Chequeo de consistencia entre planos

ENTREGA FINAL:
- Set completo de planos sellados
- Especificaciones técnicas aprobadas
- Modelo digital entregado
- Documentos listos para construcción
